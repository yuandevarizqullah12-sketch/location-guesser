# Location Guesser AI

Multi-module location guessing from images and text clues.

## Features
- Upload image (mandatory)
- Optional text clue
- Multi-source API (Google Places, OSM)
- Image processing with OpenCV
- LLaMA integration (optional)
- Smart scoring system

## Environment Variables
- `GOOGLE_MAPS_API_KEY`
- `GOOGLE_VISION_API_KEY` (optional)
- `LLAMA_API_KEY` (optional)

## Deployment
Deployed on Vercel
